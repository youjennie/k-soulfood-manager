K-Soulfood 관리 가이드북

📋 필수 관리 사항

미국 캘리포니아 Buena Park에서 한국식 포차를 운영하면서 반드시 관리해야 할 인허가, 직원관리, 운영 사항들을 추적합니다.

  • 작업 목록 — 긴급도순으로 정렬. 체크박스로 완료 표시, 날짜 관리 가능
  • 캘린더 — 면허 만료일, 점검일 등을 캘린더에서 한눈에 확인
  • AI 에이전트 — 오른쪽 채팅으로 인허가/노동법/세금 실시간 상담
빨간 글로우가 뜨는 카드는 마감 14일 이내! 최우선으로 처리하세요.

📷 SNS 콘텐츠

매장 SNS 콘텐츠를 기획, 예약, 관리합니다.

  • 월별 추천 — 현재 월에 맞는 콘텐츠 아이디어를 자동 추천
  • 콘텐츠 등록 — 플랫폼별(IG, TikTok, YouTube, Naver) 게시 예약
  • 마케팅 에이전트 — AI가 캡션, 해시태그, 게시 타이밍을 실시간 제안
추천 카드를 클릭하면 AI에게 자동으로 상세 가이드를 요청합니다.

📈 매출 분석

K SOUL FOOD POS 시스템의 CSV 데이터를 업로드하여 매출을 분석합니다.

  • Revenue Report — 매출 요약, 카테고리별 분석, 음료/메뉴 상세
  • Line Items — 주문별 분석, 서버 성과, 시간대 분석, 테이블 회전율
  • CSV 업로드 — 상단 "CSV 업로드" 버튼으로 파일 가져오기
Revenue Report와 Line Items 탭을 전환하며 다른 관점의 분석을 볼 수 있습니다.

📅 통합 캘린더

컴플라이언스 마감일, SNS 게시 예약, 직접 추가한 일정을 모두 한 캘린더에서 관리합니다.

  • 빨간 점이 있는 날짜 = 등록된 일정 있음
  • 날짜를 클릭하면 해당일 일정 확인 + 새 일정 추가 가능

⚙ 설정

AI 에이전트 설정과 데이터를 관리합니다.

  • API 키 — Anthropic API 키를 직접 입력하여 AI 에이전트 사용
  • 서버/직접 모드 — 서버 경유 또는 브라우저 직접 호출 선택
  • 데이터 초기화 — 채팅 기록 또는 전체 데이터 리셋

💡 단축키 & 팁

  • 모든 데이터는 브라우저 localStorage에 저장 (서버 전송 없음)
  • 다크/라이트 모드는 사이드바 하단 토글 또는 설정에서 전환
  • 모바일에서는 좌상단 메뉴 버튼으로 사이드바, 하단 채팅 버튼으로 AI 접근
